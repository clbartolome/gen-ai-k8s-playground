Gen AI Playground — deploy roadmap
==================================

Namespace
---------
- Single namespace for the whole stack: gen-ai-playground
- Simpler for demos, RBAC, and one ArgoCD Application
- Split into multiple namespaces later only if platform (operators) needs isolation

Labels
------
- demo=gen-ai-k8s-playground        → select all demo resources
- app.kubernetes.io/part-of=gen-ai-playground
- app.kubernetes.io/name=agent|chat → per component

  oc get all,route -l demo=gen-ai-k8s-playground -n gen-ai-playground
  oc delete all,route -l demo=gen-ai-k8s-playground -n gen-ai-playground

OpenShift SCC
-------------
- Deployments must NOT set runAsUser/fsGroup to 1001 (restricted-v2 assigns
  a UID from the namespace range, e.g. 1000740000+)
- UBI images use chmod g=u so any assigned UID works
- hostUsers: false required for restricted-v3
